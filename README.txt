RR-Tree version 0.9

The following publicly-available libraries are required to build the source:
1) JOpt Simple (http://jopt-simple.sourceforge.net/)
2) JUnit (http://www.junit.org/)
3) XXL (http://www.xxl-library.de/)

To use them, check the Ant build files to see where to extract their distributions.  The exact versions used also can be found there.
